Advanced programming skills for website design. Students will explore dynamic content development through JavaScript and related languages to create interactive websites. Students will create web pages utilizing the most current techniques in web development. The curriculum in this class emphasizes an integrated and creative approach to web programming through lectures, discussion, and coding practices. It is designed to give students an introduction to the tools and techniques used by professional web designers and programmers.
